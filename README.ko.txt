한국전력(KEPCO) 가정용 전기요금을 자동으로 계산하는 Homey 앱입니다.

kWh 사용량을 입력하면 한전의 요금체계(누진제, 하계/동계요금, 할인 등)에 따라 청구금액을 자동으로 계산합니다.

주요 기능:
- 가정용 저압/고압 요금 계산
- 계절별 누진 요금 자동 반영
- 대가족 할인, 복지 할인 지원
- 예상 사용량 및 예상 요금 계산
- Homey Flow를 통한 자동화
- Homey Insights를 통한 요금 추이 확인
- 요금 데이터 자동 업데이트 (GitHub에서 24시간마다 갱신)

Homey Python SDK 기반, 원본: Home Assistant용 kwh_to_won (dugurs)
